# report-coffee

CLI-скрипт для построения отчёта `median-coffee` по одному или нескольким CSV-файлам.
Отчёт считает медиану трат на кофе (`coffee_spent`) по каждому студенту за все переданные файлы и выводит таблицу в консоль, отсортированную по убыванию медианы.

## Пример запуска

```bash
python main.py --files math.csv physics.csv programming.csv --report median-coffee
```

или через bat-обёртку:

```bat
run_report.bat
```

## Bat-скрипты

- `run_report.bat` — обычная версия. Запускает `main.py` из этого репозитория и автоматически подставляет все `*.csv` из текущей папки.
- `run_report_portable.bat` — portable-версия. Её можно скопировать в любую папку с CSV-файлами и запустить там, даже без структуры проекта: расчёт медианы выполняется внутри PowerShell-скрипта.

## Тесты

Запуск:

```bash
pytest -q
```

Что покрыто тестами:

- успешный сценарий построения отчёта через CLI;
- ошибка при неизвестном `--report`;
- ошибка при несуществующем файле;
- ошибка при невалидном значении `coffee_spent`;
- ошибка при отсутствии обязательной колонки `coffee_spent`;
- ошибка при пустом поле `student`;
- объединение данных из нескольких файлов через `--files`;
- корректный расчёт медианы и сортировка по убыванию в отчёте.

## Параметры

- `--files` — список путей к CSV-файлам.
- `--report` — имя отчёта (в этом задании: `median-coffee`).
